# cmk-oposs_bgp_mon

A Checkmk extension for monitoring BGP sessions via the OPOSS BGP Monitor.

## Overview

This project provides integration between Checkmk and BGP monitoring capabilities, allowing you to track BGP session states and metrics within your Checkmk monitoring infrastructure.

## Components

- **Agent Plugin**: `agent_oposs_bgp_mon` - Collects BGP session data
- **Check Plugin**: `oposs_bgp_mon_sessions` - Processes and evaluates BGP session metrics

## Development

This is a Checkmk MKP (Checkmk Extension Package) that extends Checkmk's monitoring capabilities for BGP sessions.
